## HOW TO USE THIS FEATURE

You need to create a graph as a json file like THIS

```
{
  "A": {"B": 6, "C": 7, "F": 5},
  "B": {"C": 8, "D": 9, "F": 4},
  "C": {"D": 10, "E": 11, "F": 3},
  "D": {"A": 11, "E": 10, "F": 2},
  "E": {"F": 1},
  "F": {"A": 12, "B": 13, "C": 14, "D": 15, "E": 16}
}
```

Then enter Start Node and End Node
